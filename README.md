## 🚀 Projects

### 🗣️ [L/R Master](https://github.com/soramk/lr-app)
**AI Powered English Pronunciation Training App**

AI搭載型の英語発音学習アプリケーション。日本人英語学習者にとって最も困難とされる「L」と「R」の発音・聞き分けを克服するためのWebアプリです。

- **AIによる発音判定**: Google Gemini / OpenAI / Web Speech API を活用
- **音声可視化**: 波形・声紋・スペクトルでの分析
- **ゲームモード**: ブリッツ、早口言葉、フォルマントゲームなど
- **学習記録**: 統計ダッシュボードとランクシステム

[View Repository](https://github.com/soramk/lr-app)

### 🔢 [Numbers3](https://github.com/soramk/numbers3)
**AI-Powered Lottery Number Prediction Tool**

GitHub Pagesで運用するナンバーズ3の予測ツール。GitHub Actionsを利用してPythonで高度な分析を行い、14の予測手法と多様な統計分析を組み合わせて予測結果を生成します。

- **14の予測手法**: カオス理論、マルコフ連鎖、ベイズ統計、機械学習（RandomForest、XGBoost、LightGBM、LSTM、HMM）など
- **高度な分析**: 相関分析、トレンド分析、クラスタリング、ウェーブレット解析、PCA、t-SNE、ネットワーク分析など
- **AI統合分析**: Google Gemini APIによる詳細分析と統合予測の生成
- **自動更新**: GitHub Actionsによる毎日の自動データ取得と分析実行
- **可視化**: Chart.jsによる位相グラフと分析結果の可視化

[View Repository](https://github.com/soramk/numbers3)
