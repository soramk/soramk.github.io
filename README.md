## プロジェクト

### [L/R Master](https://github.com/soramk/lr-app)

AI搭載型の英語発音学習アプリケーション。日本人英語学習者にとって最も困難とされる「L」と「R」の発音・聞き分けを克服するためのWebアプリです。

**主な機能:**
- AIによる発音判定（Google Gemini / OpenAI / Web Speech API）
- 音声可視化（波形・声紋・スペクトル）
- ゲームモード（ブリッツ、早口言葉、フォルマントゲームなど）
- 学習記録・統計ダッシュボード
- ランクシステム
- その他多数の拡張機能

詳細は [lr-app リポジトリ](https://github.com/soramk/lr-app) をご覧ください。
